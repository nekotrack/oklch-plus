Methods

Oklch+ — Color Difference Metric

L′ = L^α
C′ = C^n / (C^n + σ^n)   ← Naka-Rushton compression
h′ = h

ΔE  = √(ΔL′² + Δa′² + Δb′²)

Optimal:  α=0.73  n=0.87  σ=0.34
STRESS:   29.10 on COMBVD (3,813 pairs · 6 datasets)

Hue Modulation — Singularity Resolution

w_h(C) = C^0.87 / (C^0.87 + 0.34^0.87)

C → 0  :  w_h → 0  →  Oklab linear  (hue contribution = 0)
C ≫ σ  :  w_h → 1  →  OKLCH polar  (full hue control)

No additional parameters — reuses Oklch+ chroma params.
Differentiable and continuous at all points.
