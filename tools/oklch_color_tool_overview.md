# 時彩 — 概要説明

## 一言で言うと

情景（Hue）× 時間帯（L/C）の 2 軸選択で、AI ツールに渡せるカラーシステムを自動生成する Web ツール。

---

## 背景

Web デザインの色選びが難しい理由のひとつは、RGB や HSL が人間の感覚と対応していないからです。

OKLCH は「明るさ（L）・鮮やかさ（C）・色の種類（h）」の 3 軸が知覚と一致する色空間で、スライダーの操作が「明るくしたい」「青寄りにしたい」という感覚語と直接対応します。

このツールは、その OKLCH の直感性と、知覚均等色差モデル **Oklch+**（L^α・Naka-Rushton 変換による C 圧縮）の研究成果を、初心者でも使える色選びツールとして実装したものです。

---

## 主な機能

### カラーパレット生成

情景（6種）で Hue を、時間帯（6種）で L・C を決める 2 軸選択でベースカラーが確定します。
Primary 1色を中心に 11色のカラーシステムを自動生成します。

**情景（Hue 軸）**

| 情景 | H | 色の方向 |
|---|---|---|
| 茜・緋 | 25° | 赤橙 |
| 柿・麦 | 60° | 黄 |
| 柳・翠 | 160° | 緑 |
| 空・海 | 225° | 青 |
| 露・藍 | 264° | 青紫 |
| 藤・宵 | 315° | 紫 |

**時間帯（L/C 軸）**

| 時間帯 | L | C | 語 |
|---|---|---|---|
| 深夜 | 0.28 | 0.07 | 暗沈 |
| 夜明け | 0.42 | 0.14 | 薄明 |
| 朝 | 0.60 | 0.18 | 清澄 |
| 正午 | 0.68 | 0.15 | 白昼 |
| 夕暮れ | 0.55 | 0.19 | 夕映 |
| 宵 | 0.38 | 0.10 | 静寂 |

時間帯はストリップ形式で表示され、選んだ情景のHueで6色の実色プレビューが並びます。
Hue スライダー・`色相 H 回転`ボタン・断面図ドラッグで随時微調整できます。

### フォント選択
- 折りたたみ式（`▶ フォント・サイズ設定を開く`）で必要時のみ展開
- 本文フォント（3択）・見出しフォント（3択）・サイズ（3択）
- 見本欄（常時表示）にリアルタイム反映

### 背景テーマ
- Dark / Light の 2 択（独立ステップとして配置）
- Dark 選択時はアプリ自体の配色がリアルタイムで変わる

### 出力
- **md 書き出し**：カラーシステム定義ファイル（CSS 変数・使い分けガイド・Typography・プロンプト例）
- **CSS コピー**：`:root { }` ブロックをクリップボードにコピー
- **プロンプト例**：AI ツールのチャット欄に貼るだけの指示文を自動生成

### 研究タブ（RESEARCH）
- L-C ガマット断面図（sRGB/P3 境界・Oklch+ グリッド、`色相 H 回転` 対応）
- OKLCH vs Oklch+ の等間隔サンプリング比較ストリップ
- 知覚補正インジケーター（補正量・ガマット余裕の可視化）

## 技術的背景

**Oklch+** は Oklab 座標系に 3 パラメータの知覚補正を加えた色差モデルです。

- 明度補正：L′ = L^α（α = 0.73）
- 彩度圧縮：C′ = Naka-Rushton 変換（n = 0.87、σ = 0.34）
- COMBVD データセット（3,813 色差ペア）で STRESS = 29.10、CIEDE2000 を超える精度

詳細は Zenn の研究記事シリーズを参照。

## 仕様・制限

| 項目 | 内容 |
|---|---|
| テーマ | Dark / Light（アプリ外観は Dark 専用、Light は md/CSS 出力と見本欄で確認） |
| 情景 | 6種（H = 25°〜315°、全色相を概ねカバー） |
| 時間帯 | 6種（L = 0.28〜0.68、C = 0.07〜0.19） |
| パレット | 11色、Primary 1色からの自動導出。汎用構成のため、用途に応じてAIに変数の読み替えを依頼することもできる |
| パラメータ | 知覚的根拠に基づくが厳密な実験値ではない |
| 推奨環境 | モダンブラウザ（Chrome 111+ / Safari 15.4+） |
